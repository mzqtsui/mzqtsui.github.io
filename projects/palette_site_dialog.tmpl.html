PaletteWebsite

HTML5 . CSS3 . Javascript . Angular . Nginx . Photoshop . Illustrator

Prior to the Christmas sale in December 2015, I revamped Palette's website to better match its polished hardware. The website redesign included some vector illustration work, new copy and significant HTML/CSS/Javascript refactoring.

Illustration

The illustration above shows the potential customer need and aims to identify the customer's major pain point: using the mouse as a one-size-fits-all solution on the UI controls is slow. Palette offers quick adjustments with precise tactile feedback, which you don't get from a mouse.

Profiles

On the old Profiles page, each item was an accordion. This made it a nightmare to navigate and find the appropriate profile for download. The new Profiles page features a searchbar and it uses an Angular model to handle its data. The new card design style is much easier to navigate, and lets the user see the most important information at a glance.
